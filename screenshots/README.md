# Evidence Screenshots

This folder contains screenshot evidence for the NorthStar Databases and Analytics Assignment.

The screenshots are used to prove that the database scripts, analytics scripts, and repository structure were tested successfully.

## Required Screenshots

The following screenshots should be added to this folder:

- `sql_tables_created.png`  
  Evidence that the SQL database and tables were created successfully.

- `sql_query_results.png`  
  Evidence showing the output of the analytical SQL queries.

- `python_output.png`  
  Evidence showing the output of `python/data_processing.py`.

- `r_analytics_output.png`  
  Evidence showing the output or chart generated by `r/analytics.R`.

- `mongodb_queries.png`  
  Evidence showing MongoDB documents, queries, aggregation, or index creation.

- `github_repository_structure.png`  
  Evidence showing the completed GitHub repository structure.

## Purpose of Evidence

These screenshots support the final submission by showing that:

- the SQL scripts were tested,
- the Python script was executed,
- the R analytics script was executed,
- the MongoDB queries were tested,
- and the GitHub repository was organised correctly.
